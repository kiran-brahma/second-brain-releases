# Second Brain — Releases

This repository hosts releases for [Second Brain](https://github.com/kiran-brahma/personal-rag-tauri), a local AI knowledge base built with Tauri.

## Download

Go to [Releases](https://github.com/kiran-brahma/second-brain-releases/releases) and download the latest `.dmg` for macOS.

## Installation

1. Download the `.dmg` file from the latest release.
2. Open it and drag **Second Brain.app** to your Applications folder.
3. On first launch, macOS may block the app. Right-click the app → **Open** → **Open** to bypass Gatekeeper.

> The app is not notarized with an Apple Developer certificate. This one-time override is required on first run only. Subsequent updates install automatically inside the app.

## Updates

Second Brain checks for updates on every launch. When a new version is available, a banner appears in the bottom-right corner. Click **Install** — the app downloads, applies the update, and relaunches.

## Source

Source code is in a private repository. This repo exists solely to serve release artifacts and the update manifest (`latest.json`) to the built-in updater.
